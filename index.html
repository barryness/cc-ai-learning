关于本项目

学习理念

  • 所有概念必须配可运行 Demo
  • 从问题出发（Why → What → How）
  • 渐进式深入（Phase 1 → 2 → 3）
  • 每个模块独立成体系

项目规范

  • Python 3.10+，标准库优先
  • 教学指南 + Demo + 交互网页
  • 代码注释中文，标识符英文
  • 深色主题统一视觉风格
